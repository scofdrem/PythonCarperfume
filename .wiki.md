# Project Summary
The project is a luxury perfume e-commerce platform that allows users to explore, purchase, and manage a diverse range of perfume products. It features an intuitive admin panel for product management and user engagement, enhancing the shopping experience across various brands and categories. Recent updates have streamlined the admin capabilities by enabling dynamic brand management within product forms and removing unnecessary brand management features.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic, including endpoints for user account management and feedback email configurations. Recent updates focus on improving database connection handling and enhancing product-related functionalities.
- **Frontend**: Provides a user-friendly interface for product browsing and a streamlined admin panel. Recent changes include a dynamic brand selection in product management, simplifying the admin experience.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
