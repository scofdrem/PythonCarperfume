# Project Summary
The project is an upscale e-commerce platform specializing in luxury perfumes, aimed at delivering a premium shopping experience. It features a sophisticated admin panel for managing brands, products, and site content. Recent enhancements have focused on backend data management, frontend user interface improvements, and the integration of cloud storage for image uploads in the banner management section to ensure persistence across app redeployments.

# Project Module Description
- **Backend**: Handles user authentication, database connections, and core business logic, with endpoints for managing user accounts, inquiries, and site content. The `site_content` model has been updated to support a new `Banner` structure for enhanced marketing capabilities.
- **Frontend**: Provides an intuitive interface for users to browse products and manage site content. The admin panel includes functionalities for managing dynamic banners. Recent modifications include improved error handling and cloud storage integration for banner images. The navigation has been updated to remove the "Войти" button and hide the "Админ" tab.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS
- **Cloud Storage**: Atoms Cloud Object Storage

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
