# Project Summary
The project is a luxury perfume e-commerce platform that enables users to explore, purchase, and manage a wide variety of perfume products. It features an intuitive admin panel for product management and user engagement, enhancing the shopping experience across diverse brands and categories. Recent updates have added full CRUD (Create, Read, Update, Delete) functionalities for managing brands and categories, further streamlining the admin capabilities.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic, now including new endpoints for user account management and feedback email configurations.
- **Frontend**: Provides a user-friendly interface for product browsing and an enhanced admin panel with full CRUD controls for managing brands and categories, in addition to existing product management features.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
