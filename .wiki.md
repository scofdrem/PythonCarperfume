# Project Summary
This project is a web application designed to showcase and sell luxury perfumes, providing users with an elegant browsing experience. It features a visually appealing interface with product categories, detailed product information, and a seamless shopping experience, including an age-based filtering system for products. Recently, the application has been updated to reflect changes in currency and regional references.

# Project Module Description
The project consists of several functional modules:
- **Homepage**: Displays featured products and categories.
- **Product Catalogue**: A filterable grid showcasing available products, enhanced with an age-range filter.
- **Blog Section**: Contains articles about perfumes and related topics.
- **Reusable UI Components**: A library of UI elements like buttons, forms, and dialogs for consistent design across the application.
- **Navigation**: Includes a header with search functionality, a Log In button for admin access, and a footer with contact information and social links.
- **Footer**: Updated to reflect regional changes and currency.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **components.json**: Metadata for the UI components.
- **eslint.config.js**: Configuration for ESLint to maintain code quality.
- **index.html**: Entry point for the web application.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **prerender/**: Contains scripts for generating static content for the blog.
- **public/**: Static files like images and favicon.
- **src/**: Main source directory containing application logic, components, and styles.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and bundling.
- **PostCSS**: Tool for transforming CSS with JavaScript plugins.

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```
   npm install
   ```
2. Build the project:
   ```
   npm run build
   ```
3. Run the development server:
   ```
   npm run dev
   ```
