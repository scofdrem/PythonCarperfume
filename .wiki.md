# Project Summary
The project is an upscale e-commerce platform specializing in luxury perfumes, designed to provide a premium shopping experience. It features a sophisticated admin panel for managing brands, products, and site content. Recent updates have focused on enhancing backend data management and improving the frontend user interface for better engagement, including the introduction of dynamic banners to replace static cards. The latest changes address compatibility issues with legacy data structures.

# Project Module Description
- **Backend**: Manages user authentication, database connections, and core business logic. It includes endpoints for managing user accounts, inquiries, and site content. The `site_content` model has been updated to support a new `Banner` structure for enhanced marketing capabilities.
- **Frontend**: Provides an intuitive interface for users to browse products and manage site content. The admin panel includes functionalities for managing banners, allowing for dynamic content display on the homepage. Recent modifications include improved error handling for data loading and component rendering.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
