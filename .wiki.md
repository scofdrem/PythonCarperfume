# Project Summary
The project is a luxury perfume e-commerce platform that allows users to explore, purchase, and manage a wide variety of perfume products. It features an intuitive admin panel for product management, enhancing user engagement, and streamlining the shopping experience across multiple brands and categories.

# Project Module Description
- **Backend**: Handles authentication, database interactions, and business logic.
- **Frontend**: Offers a user-friendly interface for product browsing and an admin panel for product management.
- **Admin Panel**: Enables administrators to efficiently manage products, brands, categories, and site content.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
