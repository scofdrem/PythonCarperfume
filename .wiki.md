# Project Summary
The project is a luxury perfume e-commerce platform designed for users to explore, purchase, and manage a diverse range of perfume products. It includes an intuitive admin panel that facilitates the management of product listings, enhancing user engagement and streamlining the shopping experience across various brands and categories.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic.
- **Frontend**: Provides a user-friendly interface for product browsing and an admin panel for product management.
- **Admin Panel**: Empowers administrators to oversee products, brands, and categories efficiently.

# File Description Inventory
- **app/backend/**: Contains backend-related files including API routes, models, and services.
- **app/frontend/**: Contains the frontend application code, including components, pages, and assets.
- **start_app_v2.sh**: Script to initialize and start the application.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
