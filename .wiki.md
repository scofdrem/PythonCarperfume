# Project Summary
The project is a luxury perfume e-commerce platform aimed at delivering an exceptional shopping experience. It offers a diverse collection of perfume products and features an intuitive admin panel for efficient product management. Recent updates have focused on enhancing brand management and optimizing user engagement through a revamped feedback inquiry system.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic. It includes endpoints for user account management and feedback submissions. The `InquiryRequest` model has been updated to allow optional customer names and introduced new fields for improved inquiry handling.
- **Frontend**: Provides a user-friendly interface for product browsing and features a dynamic admin panel. The `InquiryModal` component has been enhanced for better user feedback management.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
