# Project Summary
The project is a luxury perfume e-commerce platform that enables users to explore, purchase, and manage a wide range of perfume products. It includes an intuitive admin panel for product management and user engagement, enhancing the shopping experience across various brands and categories. Recent updates have improved the admin panel by adding bulk edit functionality for product management.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic, including endpoints for user account management and feedback email configurations. Recent updates focus on improving database connection handling and enhancing product-related functionalities.
- **Frontend**: Provides a user-friendly interface for product browsing and an enhanced admin panel. Recent changes include the addition of bulk edit functionality, allowing for efficient management of multiple products at once.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
