# Project Summary
The project is a luxury perfume e-commerce platform designed to provide a premium shopping experience. It features a wide array of perfume products and includes an intuitive admin panel for efficient brand and product management. Recent enhancements have focused on reinstating brand management capabilities and optimizing user engagement through improved feedback systems.

# Project Module Description
- **Backend**: Responsible for authentication, database interactions, and core business logic. It includes endpoints for user account management and feedback submissions. The `InquiryRequest` model allows optional customer names and has new fields for better inquiry handling.
- **Frontend**: Offers a user-friendly interface for browsing products and features an enhanced admin panel. The admin panel now includes a "Brands" management tab, allowing for dynamic brand management directly linked to the product list.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
