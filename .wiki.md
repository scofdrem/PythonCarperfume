# Project Summary
The project is an upscale e-commerce platform specializing in luxury perfumes. It aims to deliver a premium shopping experience through a diverse selection of products and a sophisticated admin panel for managing brands, products, and site content. Recent updates have focused on enhancing backend data management and improving the frontend user interface for better engagement.

# Project Module Description
- **Backend**: Manages user authentication, database connections, and core business logic. It includes endpoints for managing user accounts, handling inquiries, and managing site content. The `site_content` model has been updated to include a nullable `content_key` and a new `content_value` field for enhanced data handling.
- **Frontend**: Provides an intuitive interface for users to browse products and manage site content. The admin panel now features improved functionalities for managing brands and categories, with a focus on seamless data persistence through API integration.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
