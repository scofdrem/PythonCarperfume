# Project Summary
The project is a premium e-commerce platform specializing in luxury perfumes, delivering an upscale shopping experience. It features a sophisticated admin panel for managing brands, products, and site content. Recent enhancements have focused on backend data management, improving the frontend user interface, and integrating email notifications for inquiries, alongside persistent image uploads through cloud storage for banner management.

# Project Module Description
- **Backend**: Manages user authentication, database connections, and core business logic. It includes endpoints for user accounts, inquiries, and site content management. The `site_content` model has been updated for better marketing capabilities, and a new inquiry submission feature now includes email notifications to the admin.
- **Frontend**: Offers an intuitive interface for users to browse products and manage site content. The admin panel has been enhanced to support CRUD operations for products and categories, with improvements in error handling, data persistence, and dynamic banners.

# File Description Inventory
- **app/backend/**: Contains backend files, including API routes, models, and services, with updates for enhanced inquiry handling and email notifications.
- **app/frontend/**: Contains the frontend application code, featuring dynamic banners and improved admin panel functionalities for product and category management, including data persistence to the backend.
- **start_app_v2.sh**: Script to initialize and start the application.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS
- **Cloud Storage**: Atoms Cloud Object Storage

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
