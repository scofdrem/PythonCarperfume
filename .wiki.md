# Project Summary
The project is a luxury perfume e-commerce platform aimed at delivering an exceptional shopping experience for users. It offers a wide variety of perfume products, features an intuitive admin panel for effective product management, and facilitates seamless user interactions. Recent updates have enhanced brand management and introduced a feedback inquiry system to improve customer engagement.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic. It includes endpoints for user account management and feedback submissions. Recent updates have improved database handling and introduced an inquiry submission feature.
- **Frontend**: Provides a user-friendly interface for browsing products and a dynamic admin panel. The latest update includes a new InquiryModal component and an inquiry button on product cards for user feedback.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
