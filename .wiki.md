# Project Summary
The project is a luxury perfume e-commerce platform designed to provide users with an exceptional shopping experience for a diverse range of perfume products. It features a streamlined admin panel for efficient product management and user engagement, allowing for seamless exploration and purchase across various brands. Recent enhancements have introduced dynamic brand management, improving usability and maintaining brand consistency across the platform.

# Project Module Description
- **Backend**: Handles authentication, database interactions, and business logic, including endpoints for user account management and feedback configurations. Recent updates have improved database connection handling and product-related functionalities.
- **Frontend**: Offers a user-friendly interface for product browsing and an enhanced admin panel. Updates include the implementation of a dynamic brand management system that generates a brand list from product data, ensuring real-time updates across the platform.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
