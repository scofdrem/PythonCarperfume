# Project Summary
The project is a luxury perfume e-commerce platform that allows users to explore, purchase, and manage a diverse range of perfume products. It features an intuitive admin panel for product management and user engagement, streamlining the shopping experience across various brands and categories. Recent updates have introduced a comprehensive user account management system for admins, enabling them to manage user roles and settings effectively.

# Project Module Description
- **Backend**: Handles authentication, database interactions, and business logic. It includes new endpoints for user account management and feedback email configurations.
- **Frontend**: Offers a user-friendly interface for product browsing and an enhanced admin panel that now includes user management functionalities alongside existing product management features.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
