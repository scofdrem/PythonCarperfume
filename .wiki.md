# Project Summary
The project is a luxury perfume e-commerce platform that allows users to browse, purchase, and manage perfume products. It features an admin panel for managing product listings, including creation, editing, and publishing functionalities, aimed at enhancing the user experience in navigating through various products and brands.

# Project Module Description
- **Backend**: Handles authentication, database interactions, and business logic.
- **Frontend**: User interface for browsing products, including an admin panel for product management.
- **Admin Panel**: Allows admins to manage products, brands, and categories.

# File Description Inventory
- **app/backend/**: Contains all backend-related files including API routes, models, and services.
- **app/frontend/**: Contains the frontend application code, including components, pages, and assets.
- **start_app_v2.sh**: Script to initialize and start the application.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
