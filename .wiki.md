# Project Summary
The project is a luxury perfume e-commerce platform that enables users to explore, purchase, and manage a diverse range of perfume products. It features an intuitive admin panel for product management, enhancing user engagement, and streamlining the shopping experience across multiple brands and categories. Recent updates have introduced enhanced admin account management functionalities, including email and password updates, as well as a feedback email configuration.

# Project Module Description
- **Backend**: Manages authentication, database interactions, and business logic, including new endpoints for admin account management and feedback email settings.
- **Frontend**: Provides a user-friendly interface for product browsing and an admin panel for product management, now updated to include account and settings tabs.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS

# Usage
To set up the project:
1. Install dependencies:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install`
2. Build the project:
   - For backend: Run the backend server using a command like `uvicorn app.backend.main:app --reload`
   - For frontend: Run the build command `pnpm run build`
3. Start the application using `./app/start_app_v2.sh`
